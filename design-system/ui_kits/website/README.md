# BRALARMSEG — Website UI Kit

## Overview
High-fidelity interactive prototype of the BRALARMSEG institutional website.
Based on: https://www.bralarmseg.com.br/lp/ (scraped May 2026) + logo asset.

## Files
- `index.html` — Full interactive prototype (homepage → product page click-through)
- `Header.jsx` — Sticky nav with topbar
- `Hero.jsx` — Hero section with headline + CTA
- `ProductCard.jsx` — Product card component
- `ProductGrid.jsx` — Products section grid
- `ApplicationsSection.jsx` — Segment tiles grid
- `StatsSection.jsx` — Stat callout strip
- `TestimonialsSection.jsx` — Client testimonials
- `Footer.jsx` — Footer with contact info

## Design width
1280px content max-width, responsive down to 768px.

## Notes
- No Figma file was provided — recreated from live website scrape
- Logo used as CSS text lockup (matching brand letterforms) since PNG has white background
- WhatsApp CTAs are non-functional placeholders in prototype
